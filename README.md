# Calculadora JavaScript

🧮 Calculadora em JavaScript

Um projeto simples e elegante de uma calculadora desenvolvida com HTML, CSS (Bootstrap) e JavaScript puro, com o objetivo de praticar conceitos fundamentais da linguagem.
Interface baseada em card utilizando Bootstrap, com inputs intuitivos e seleção de operações matemáticas.

🚀 Funcionalidades
- ✅ Soma
- ✅ Subtração
- ✅ Multiplicação
- ✅ Divisão
- ✅ Potenciação
- 🎯 Exibição dinâmica do resultado
- 🎨 Feedback visual com alertas (sucesso/erro)

🛠️ Tecnologias utilizadas
- HTML5
- CSS3
- Bootstrap 5
- JavaScript (Vanilla JS)

💡 Como funciona

A lógica da aplicação é baseada em:
- Captura dos valores digitados pelo usuário
- Identificação da operação escolhida
- Execução da função correspondente
- Exibição do resultado na tela

📚 Objetivo do projeto

Este projeto foi desenvolvido como estudo de JavaScript, focando em:

- Manipulação do DOM
- Estruturas condicionais
- Funções
- Eventos (onclick)
- Integração com interface (Bootstrap)

▶️ Como executar

Baixe ou clone este repositório
Abra o arquivo index.html no navegador
